## Just A Rather Very Intelligent System


This is a initial script to install and create basics when you install new linux in your pc

1. Download repository
2. Gives permissions:
```
sudo chmod +x initial.sh
```
3. RUN!
```
sudo ./initial.sh>
```

### What do you install?

1. Terminator
2. wget
3. Chrome
4. Visual Studio Code
